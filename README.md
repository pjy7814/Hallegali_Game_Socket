### 시나리오
1. 3명의 게임 참가자가 정상적으로 게임에 입장
2. 사용자가 card 입력 (정상/비정상)
3. 사용자가 bell 입력 (정상/비정상)
4. 사용자의 카드 수가 부족할 때 쫓겨남
5. 남은 사용자의 수가 1명일 때, 우승자 선정
